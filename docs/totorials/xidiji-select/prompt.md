# 整体要求：
- 使用翻页组件库，样式参考已有项目
- 首先完成 index.vue，并加入路由 ./src/router/index.ts
- 采用 slide-{序号}-{名称}.vue 的命名规则
- 只使用共享组件
- 所有页面使用背景组件功能（不要带背景色）

# Page 2 开场页
标题：现在的洗地机市场
展示内容：
- 群魔乱舞，参数多到让人头皮发麻
- 几万帕吸力、AI智能识别等各种天花乱坠的参数
- 引出问题：怎么挑？怎么避坑？

# Page 3 核心观点页
标题：三个核心指标
副标题：死磕三个指标，避开90%的工业垃圾
展示三个核心：
1. 搞懂洗地机的真实定位
2. 清洁三维硬指标
3. 后勤与自清洁功能

# Page 4 核心一：适用人群
标题：核心一：谁需要洗地机？
副标题：地面清洁领域的"版本之子"
展示三类人群卡片：
- 满地爬的熊孩子
- 养了掉毛怪的宠物家庭
- 经常清理厨房重油污
强调：无脑冲洗地机

# Page 5 核心一：清洁原理
标题：洗地机为什么强？
副标题：三合一清洁方式，对付复杂家居环境的最优解
展示清洁流程：
1. 前滚刷高速洗
2. 中间强力抽污
3. 后置刮条擦干
注意：怕水实木地板除外

# Page 6 核心二：吸力指标
标题：硬指标一：吸力大小
副标题：虚假宣传重灾区
展示关键数据：
- 15000Pa = 及格线（低于这个连饼干渣都吸不干净）
- 20000Pa = 推荐（重油污/不频繁打扫）
- 注意：超大吸力 = 高噪音

# Page 7 核心二：防缠绕
标题：硬指标二：防缠绕系统
副标题：洗地机易不易用的分水岭
展示核心技术卡片：
- 梳齿设计
- 主动切割技术
- 毛发分离机制
避免现象：滚刷缠成毛线团

# Page 8 核心二：死角覆盖
标题：硬指标三：平躺与贴边
副标题：传统拖把够不到的才是痛点
展示两大要点：
1. 真平躺：180°平躺 + 机身≤12cm（钻进家具底部）
2. 三侧贴边：≤5mm（墙角不留黑线）

# Page 9 核心三：热风烘干
标题：后勤一：自清洁烘干
副标题：用完还得手洗滚刷，买来绝对吃灰
展示对比卡片：
- ❌ 离心甩干/自然晾干 → 梅雨季发霉发臭
- ✅ 热风烘干 → 2小时内，70℃热水洗

# Page 10 核心三：续航水箱
标题：后勤二：续航配置
副标题：根据户型算明白账
展示大户型（120平+）配置：
- 电池：≥4000mAh
- 续航：≥40分钟
- 清水箱：≥800ml
避免：拖一半没电没水的尴尬

# Page 11 核心三：助力系统
标题：后勤三：助力轮设计
副标题：女生、老人刚需
展示功能：
- 双向助力万向轮
- 单手推拉不费腰

# Page 12 三大陷阱
标题：三大恶心陷阱
副标题：厂商的文字游戏坑多着呢
展示三个陷阱卡片：
1. 直播间套路：半价秒杀 = 老款清库存，糖水演示，带瓶酱油去试
2. 耗材刺客：滚刷大几十上百，滤网不能水洗，羊毛出在羊身上
3. 假平躺：平躺后吸力下降、水箱漏水

# Page 13 总结页
标题：选购要点总结
展示四个核心：
1. 吸力 ≥15000Pa
2. 防缠绕系统
3. 热风烘干
4. 真平躺 + 三侧贴边
避坑：直播间清库存、耗材刺客
结论：拖得干净、用得爽的地面清洁神器
