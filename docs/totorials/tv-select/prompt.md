# Page 3 核心一：屏幕技术
标题：核心一：看屏幕技术
副标题：2026年的版本之子
展示三种屏幕类型对比：
- Mini LED（推荐）：亮度高、控光好、无烧屏担忧
- OLED：极致画质，但价格贵、可能烧屏
- 普通 LED：价格便宜，但画质一般
强调 Mini LED 是大多数家庭的最优解

# Page 4 核心二：画质三维硬指标
标题：核心二：画质三维硬指标
副标题：想要画面通透、色彩真实，必须卡死这三点
展示三个指标卡片：
1. 峰值亮度：至少1000尼特以上
2. 背光分区：百级起步，千级优秀
3. 色域覆盖：DCI-P3 必须90%以上，最好95%

# Page 5 核心三：芯片和接口
标题：核心三：看大脑和接口
展示两个要点：
1. 芯片：认准联发科 Pentonic 700 (MT9653) 或同级别高端芯片
2. 接口：HDMI 2.1 必须是满血版 48Gbps，支持 VRR 和 ALLM

# Page 6 三大陷阱防坑指南
标题：三大恶心陷阱
副标题：厂商的文字游戏坑多着呢
展示三个陷阱卡片：
1. 假 HDMI 2.1：带宽只有18Gbps，本质是 HDMI 2.0 换马甲
2. 假高刷：DLG/HSR 技术，牺牲分辨率换假高刷
3. 开机广告和系统封闭：严重影响日常体验

# Page 7 总结页
标题：选购要点总结
展示核心要点：
1. 认准 Mini LED
2. 盯紧亮度、分区、色域三大硬指标
3. 避开假接口和假高刷的坑
结论：画质顶、用得爽的好电视
