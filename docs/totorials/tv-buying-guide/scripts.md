# 第2页：
我们先来看看大家买电视时最常遇到的困惑

- 参数表看得头晕，什么HDR、杜比视界、MEMC，完全搞不懂

- 型号字母数字一大串，怎么看都像乱码
- 最怕的就是买完第二天就降价，或者发现买贵了

这些困惑的本质是什么？

信息不对称。商家用专业术语包装，消费者又缺乏判断依据。

# 第3页：
先说第一个问题：买多大？

很多人觉得越大越好，其实不是。

电视尺寸要根据观看距离来选：

- 如果你家沙发到电视柜距离是2到2.5米，55英寸最合适
- 2.5到3米，选65英寸
- 3到3.5米，可以上75英寸
- 超过3.5米，85英寸起步

记住一个简单公式：距离（米）× 25 = 建议尺寸（英寸）

太大了伤眼，太小了看着累，合适才是最好的。

# 第4页：
屏幕类型，是目前选购电视最重要的决策点。

主要有三种：

LCD，也叫液晶电视，技术成熟，价格实惠，大部分入门到中端产品都是这个。缺点是对比度一般，暗部细节容易糊。

Mini LED，可以理解为LCD的升级版。用了很多很多小灯珠做背光，亮度高、对比度好，是现在中高端的主流选择。

OLED，每个像素自发光，黑色纯、对比度无限大，画质最好。但价格贵，而且长时间显示固定画面可能烧屏。

简单总结：预算有限选LCD，追求画质选Mini LED，预算充足要极致选OLED。

# 第5页：
分辨率和刷新率，这两个参数经常被混淆。

分辨率是画面的清晰度。现在主流是4K，3840×2160像素。8K目前内容太少，没必要追。1080P只有小尺寸低端机才会用了。

刷新率是每秒画面更新的次数。60Hz是标准，看普通视频够用。120Hz看体育赛事、玩游戏体验更好。144Hz以上是高端游戏电视的配置。

如果你看电影电视剧为主，60Hz够了。如果看球赛、玩游戏，一定要上120Hz。

# 第6页：
除了屏幕，还有一些参数需要注意：

HDR——高动态范围，让画面明暗细节更丰富。要注意支持杜比视界的更好。

色域——决定颜色表现，DCI-P3 90%以上算不错。

处理器——影响系统流畅度，至少4核A55起步。

内存——运行内存3GB起步，存储空间16GB起步。

接口——如果你要接游戏主机，HDMI 2.1接口是必备的。

# 第7页：
品牌选择上，各有特点：

索尼，画质调教能力业内顶尖，但价格偏高。适合追求画质、预算充足的朋友。

三星，面板自产自销，高端QLED表现出色，但系统相对封闭。

LG，OLED电视的首选，C系列性价比很高。

TCL和海信，国产双雄，Mini LED领域性价比突出，是大多数人最稳妥的选择。

小米，智能生态完善，价格亲民，适合追求性价比的用户。

# 第8页：
最后聊聊价格，不同预算怎么选：

3000元以下，适合出租房、老人房，基本功能都有，画质别有太高期待。

3000到6000元，这是竞争最激烈的区间，能买到不错的65英寸Mini LED电视，性价比最高。

6000到10000元，画质明显提升，可以买到75英寸以上的好产品，适合对画质有要求的家庭。

10000元以上，就是各家的旗舰了，画质、做工、体验都顶级，适合发烧友和预算充足的用户。

# 第9页：
说几个避坑的要点：

不要只看宣传参数。峰值亮度、对比度这些，很多都是理想值，实际效果要看评测。

注意系统广告。有些品牌开机广告、系统广告很多，影响体验，买之前了解清楚。

关注售后。电视是大件，出问题维修麻烦，选择售后网点多的品牌更放心。

警惕虚假HDR。有些低端机也说支持HDR，其实效果和没有一样。

考虑实际场景。你是在客厅看还是卧室？白天看多还是晚上？这些都会影响选择。

# 第10页：
总结一下，记住这三点：

第一，先定预算和尺寸。这是最基本的两条线，确定了才能往下选。

第二，看清核心参数。分辨率4K、刷新率按需选、HDR看有没有杜比视界。

第三，选择靠谱品牌。主流品牌品控和售后都有保障。
