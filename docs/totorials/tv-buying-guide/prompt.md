# 电视选购指南 PPT 提示词

在 src/presentations/tv-buying-guide 内完成这个 PPT 项目

风格上请参考 design-language.md（docs/totorials/1 Skill Tree/design-language.md）

# 整体要求：
- 使用翻页组件库，样式参考 design-language-template
- 首先完成 index.vue，并加入路由 ./src/router/index.ts
- 采用 slide-{序号}-{名称}.vue 的命名规则
- 只使用共享组件
- 所有页面使用背景组件功能（不要带背景色）

---

# Page 1 标题页
参考 design-language-template 的 hero 页风格
主标题：电视选购指南
副标题：一文读懂如何挑选适合你的电视
标题使用渐变色处理

# Page 2 痛点页
标题：买电视的困惑
展示消费者选购电视时的常见痛点（3-4个卡片）：
- 参数复杂看不懂
- 型号众多挑花眼
- 价格差异大，不知道值不值
- 害怕买完就降价

# Page 3 尺寸选择
标题：多大尺寸合适？
副标题：根据观看距离选择
展示尺寸与距离的对应关系（可用表格或卡片）
- 2-2.5米 → 55英寸
- 2.5-3米 → 65英寸
- 3-3.5米 → 75英寸
- 3.5米以上 → 85英寸+

# Page 4 屏幕类型
标题：屏幕类型怎么选？
对比展示三种主流屏幕类型：
- LCD（性价比之选）
- Mini LED（画质进阶）
- OLED（极致画质）

每项列出优缺点和适用场景

# Page 5 分辨率与刷新率
标题：分辨率与刷新率
左侧：分辨率说明（1080P / 4K / 8K）
右侧：刷新率说明（60Hz / 120Hz / 144Hz）
用图标或图表展示差异

# Page 6 核心参数清单
标题：选购核心参数清单
用清单/表格形式展示：
- 分辨率：推荐4K
- 刷新率：游戏用户选120Hz+
- HDR：支持杜比视界更佳
- 接口：HDMI 2.1必备
- 系统：考虑广告问题

# Page 7 品牌推荐
标题：主流品牌推荐
用卡片展示各品牌特点：
- 索尼：画质调教出色
- 三星：面板自产，高端选择
- LG：OLED领域领先
- TCL/海信：性价比之选
- 小米：智能生态完善

# Page 8 价格区间
标题：不同预算怎么选？
按价格区间给出推荐：
- 3000元以下：入门级，满足基本需求
- 3000-6000元：主流选择，性价比高
- 6000-10000元：进阶级，画质出色
- 10000元以上：旗舰级，极致体验

# Page 9 避坑指南
标题：避坑指南
列出选购时需要避免的坑（清单形式）：
- 不要只看宣传参数
- 注意系统广告问题
- 关注售后服务
- 警惕虚假HDR
- 考虑实际使用场景

# Page 10 总结页
标题：选购要点总结
副标题：记住这三点就够了
提炼3个核心建议：
1. 先定预算和尺寸
2. 看清核心参数
3. 选择靠谱品牌

# Page 11 结束页
标题：谢谢观看
副标题：祝你选到心仪的电视
